Setup Guide

One-time 5-minute setup to enable live multiplayer.

You're seeing this because FIREBASE_READY is still set to false in the HTML file.
Step 1Go to console.firebase.google.com and sign in with Google
Step 2Click "Add project" → name it anything → create
Step 3Click the </> Web icon → register app → copy the firebaseConfig
Step 4In the left menu: Build → Realtime Database → Create database → Start in test mode
Step 5Open this HTML file in a text editor, paste your config into FIREBASE_CONFIG, set FIREBASE_READY = true
Step 6Upload the file to any host (Netlify drag-and-drop is easiest — free)
Netlify hosting (free, 1 min): Go to app.netlify.com → drag your HTML file onto the page → get a permanent URL to share.
